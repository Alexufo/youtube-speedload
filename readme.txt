Improve rendering speed pages with YouTube players.

== Description ==
Youtube SpeedLoad allow render in browser youtube players much faster! Youtube SpeedLoad replace standard embed code (<iframe src="youtube bla-bla-bla...) to thumbs from video with play button. Does not generate dependencies. You can disable this plugin any time of use.

Just click install and forget! 
== Installation ==
1.Upload the zip-file and unzip it in the /wp-content/plugins/ directory
2. Activate the plugin through the \'Plugins\' menu in WordPress

All YouTube links video like this will be affected:
